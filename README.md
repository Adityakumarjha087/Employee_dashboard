## Getting Started

1. Clone the repository:
```bash
git clone <your-repository-url>
cd my-next-app
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open (https://employee-hub-nu.vercel.app) in your browser.
